📄 Dokumentasi Tempail API

🔧 Endpoint: Buat Email

GET /api/create-email

Menghasilkan email sementara dan token.

📬 Endpoint: Ambil Semua Pesan

GET /api/messages/:token

Mengambil semua pesan dari email berdasarkan token.

📥 Endpoint: Buka Pesan

GET /api/message/:id

Mengambil detail isi pesan berdasarkan ID.

⚠️ Rate Limit

⛔ Maksimum: 30 request / menit per IP

Jika melebihi, akan muncul kode HTTP 429
